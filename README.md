Задача
Разработать сервис на Laravel 8+, который решает задачу синхронизации данных из БД с Google Таблицей (по одной модели/таблице).


Требования
Должен быть реализован CRUD интерфейс для 1 модели Eloquent (дизайн и стек фронта на ваше усмотрение)
Одно из полей в модели должно быть поле status, тип enum, возможные значения [Allowed, Prohibited]
Должна быть кнопка сгенерировать 1000 текстовых строк (с равномерным распределением значений поля статус)
Должна быть кнопка полностью очистить таблицу.
Должна быть возможность через интерфейс задать URL на документ google sheet (у документа будут права на редактирование любым
пользователем)
Система в автоматическом режиме должна выгружать/обновлять данные в указанном документе 1 раз в минуту.
Должны отправляться только строки со статусом Allowed. Для выборки используйте Local Scope.
Если статус выгрузки изменился с Allowed на Prohibited, то строка из выгрузки удаляется. Если произошло наоборот - строка
добавляется. При удалении из БД строка в таблице удаляется.
Пользователь будет писать комментарии в дополнительном столбце в google документе (это n+1 столбец по-умолчанию,
где n - кол-во столбцов в выгрузке). Необходимо, чтобы при очередном обновлении данных эти данные не затирались
и “не съезжали” относительно своей строки, если записи добавились/удалились.
В таблицу необходимо выводить все поля созданной модели.
В системе должна быть реализована консольная команда, которая:
Получает данные из google таблицы
Построчно выводит информацию об ИД модели / Комментарии из гугл таблицы в консоль
При этом в процессе в консоле отображается progressbar процесса
В команду можно передать параметр count, который ограничивает количество выводимых в консоль строк
Должен быть реализован роут /fetch, который можно вызвать из браузера.
При вызове должна быть запущена команда из п.11, и консольный вывод должен быть отображен в браузере в виде строки.
Должна быть возможность вызвать роут с параметром, чтобы задать ограничение на количество выводимых данных (см. пункт 11d).
Формат такого роута: /fetch/20
